# Drum_K (OLD version)
Analogue drum kick

## Controls are :

 - Soft/Hard attack
 - Decay
 - Tune
 - Tone
 - Distortion

### Inputs

 - Kick trigger
 - [ ] Decay CV
 - [ ] Tune CV
